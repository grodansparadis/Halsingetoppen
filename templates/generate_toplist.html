Inställningar för topplista
Hämtar senaste popularitet, följarantal och bilder från Spotify API. Rekommenderas för att få aktuell data, men tar längre tid.
Viktigt att veta
  • Uppdatering från Spotify kan ta 5-10 minuter för {{ artist_count }} artister
  • HTML-filen sparas som topplista-ÅÅÅÅ-MM-DD.html
  • Endast aktiva artister inkluderas i listan
  • Kräver aktiv internet-anslutning
Avbryt
Vad skapas?

HTML-filen kommer att innehålla:

  • Komplett topplista sorterad efter popularitet
  • Artistnamn med länkar till Spotify
  • Popularitetspoäng (0-100)
  • Antal följare
  • Artistbilder (100x100px)
  • Styling med CSS
  • Google Analytics-kod
Teknisk information
